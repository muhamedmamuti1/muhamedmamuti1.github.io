## Getting Started

### Prerequisites

- Node.js 20.19+ (or 22.12+)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd mm-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```

5. Preview production build
```bash
npm run preview
```

## Customization

### Adding Your Profile Picture

Place your profile picture in the `public` folder as:
- `profile-picture.jpg` (preferred)
- `profile-picture.png` (fallback)

### Adding Your Logo

Place your logo in the `public` folder as:
- `logo.png` (preferred)
- `logo.svg` (fallback)

### Updating Content

Edit `src/data/content.js` to update:
- Hero highlights
- Work experience
- Skills
- Contact information
